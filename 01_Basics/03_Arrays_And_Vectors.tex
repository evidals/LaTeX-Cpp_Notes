\subsection{Difference}
\begin{itemize}
    \item array - they are fixed-size collections consisting of data items of the same type
    \item vector - same as array but they can grow and shrink dynamically at execution time. 
\end{itemize}

\subsection{Local Arrays}
A static local variable in a function definition exists for the program’s duration but is visible only in the function’s body.
\subsubsection{Static Local Array}
If a static array is not initialized explicitly by you, each element of that array is initialized to zero by the compiler when the array is created. Recall that C++ does not perform such default initialization for other local variables.
